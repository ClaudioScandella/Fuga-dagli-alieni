REGOLE GENERALI PER GIOCARE:

1: quando ci si connette come client viene chiesto il proprio nome utente
2: secondo input richiesto è il nome della mappa su cui si vuole giocare
3: se esiste partita disponibile veniamo automaticamente aggiunti a quella,
	altrimenti si crea una nuova partita in automatico
4: per iniziare la partita è necessario (oltre ad essere almeno due client connessi e iscritti alla
	stessa partita) dare il comando "inizia partita"
5: una volta iniziata la partita, quando sarà il vostro turno avrete dei comandi a disposizione:
		-mossa: per effettuare il movimento
		-carta: per poter visualizzare e/o usare una carta
		-attacco: per attaccare
		-pesco oggetto: per pescare un oggetto quando ne avete la possibilità
		-passo: per passare il proprio turno, se possibile.
	questi saranno disponibili solo al giocatore che possiede il turno,
	tutti gli altri (e compreso lui stesso) avranno la possibilità di comunicare in chat con
	il comando 'chat MESSAGGIO' dove al posto di MESSAGGIO scriverete ciò che volete comunicare agli
	altri giocatori della vostra partita.
6: il resto dei comandi che potete inserire vi saranno noti tramite le istruzioni
	stampate a video che riceverete dal server
	
NOTA: per effettuare il movimento su di una scialuppa è necessario scrivere il nome
	nel formato LETTERA+NUMERO.
